    Для написания текстов

        От большого к маленькому по важности
            <h1></h1>
            <h2></h2>
            <h3></h3>
            <h4></h4>
            <h5></h5>
            <h6></h6>

        Просто для написания текста в переводе просто тег Параграфа (Обычно используется когда нада что-то переделать)
            <p></p>

        Для создания кликабельного текста
            <a href="сылка на то куда нада отправить пользователя">

    Атрибуты тегов

        src ==> Сылка на файл
        href ==> Сылка на какой-то сайт
        class ==> Для выдачи класа по которому будет связоватся htmc c css
        id ==> Для выдачи айди (Почти всегда нада для взаемодействия c javaScript)
        style ==> Выдача опридельоного атрибута с css сразу в html
        alt ==> Альтернатива чего-то (Выводет текст)
        ... (И ещё дохуя всего но используется достаточно редко. Проще говоря что написать сайт можна даже знаю только это)
